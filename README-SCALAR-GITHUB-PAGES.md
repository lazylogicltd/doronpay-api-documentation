# DoronPay API Documentation - Scalar UI

This package serves the DoronPay OpenAPI JSON using Scalar API Reference.

## Deploy to GitHub Pages

Upload these files to the root of your GitHub Pages branch/folder:

- `index.html`
- `doronpay-unified-api.github-pages.json`

In this package, the HTML is named `scalar-index.html`.
Rename it to `index.html` before deploying.
